## Quick Start
install cmake and then clone this repo  
安装cmake并克隆该仓库

```
git clone git@github.com:JNURay/DesignPatternsCPP.git
cd DesignPatternsCPP
mkdir build
cd build
cmake ..
cmake --build .
```
